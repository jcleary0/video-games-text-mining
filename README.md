## üìå Project Overview

Online customer reviews play a critical role in shaping consumer purchasing decisions and influencing product success, particularly in large digital marketplaces such as Amazon. These reviews contain vast amounts of unstructured textual data that reflect customer experiences, opinions, and satisfaction levels. Effectively analyzing this text at scale requires the application of text mining and machine learning techniques.

This project applies an end-to-end text analytics workflow to the Amazon Video Game Reviews dataset, transforming raw review text into structured, actionable insights. The analysis leverages both unsupervised and supervised learning methods to explore patterns in customer feedback and predict review characteristics based on textual content.

The primary objectives of this project are to:

- Measure similarity among customer reviews using text-based features  
- Group reviews into meaningful clusters using unsupervised learning  
- Build supervised machine learning models for review classification  
- Demonstrate how text analytics can be used to understand customer sentiment and product perception

## üß† Methods & Techniques

This project incorporates a comprehensive set of text analytics and machine learning methods, including:

- Text preprocessing (tokenization, stopword removal, normalization)
- Feature extraction using techniques such as TF-IDF
- Similarity analysis among customer reviews
- Clustering algorithms to identify groups of similar reviews
- Supervised classification models using review text and star ratings
- Model evaluation using appropriate performance metrics

Together, these techniques form a complete pipeline for analyzing large-scale customer review data.
